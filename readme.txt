You can introduce the behavior of playing sound by hovering mouse pointer over an HTML element and stopping the sound by moving the mouse pointer away from the HTML element.

== Description ==
You can introduce the behavior of playing sound by hovering mouse pointer over an HTML element in an article and stopping the sound by moving the mouse pointer away from the HTML element.


(usage of file manager) <br>
You can associate sounds with HTML elements by writing shortcodes in an article. The parts of hortcodes are not displayed when displaying an article. Also you can put shortcodes anywhere in an article.

For more details, please see the document from the following URL:<br>
http://tempspace.net/plugins/?page_id=308 <br><br>

(example)

&lt;a id="hr1" href="http://google.com"&gt;Google&lt;/a&gt; <br>
&lt;span id="hr2"&gt;ABC&lt;/span&gt; <br>
&lt;img id="hr3" src=”http://example.com/image.jpg”&gt;

[hoversound id="hr1" url="http://example.com/sound1.mp3"] <br>
[hoversound id="hr2" url="http://example.com/sound2.mp3"] <br>
[hoversound id="hr3" url="http://example.com/music1.mp3" autoload="false"]


== Installation ==

Install the plugin like usual ones. Then activate it.
